>>> API_livre


-Como funciona:

	Este projeto tem como objetivo a prática de uso de API's por meio de três exercícios, sendo este o terceiro 	deles (CEP, Temperatura e Livre). A proposta é Livre e foi escolhida a obtenção de informações de cards de YuGiOh!.
	Com a experiência obtida com os dois exercícios anteriores, este trabalho foi relativamente mais fácil de ser executado. Começando com a contrução do HTML com alguns 'input', 'button' e 'textbox', de forma simples o foco foi transferido para a construção do arquivo JS. A ideia é basicamente digitar o nome do Card(em inglês) e então a informação ser mostrada nos devidos espaços, bem como a imagem do mesmo. O processo de obtenção das informações na API foi o mais simples dos três exercícios, e o resultado foi bastante satisfatório. A parte gráfica não teve muito foco, já que a proposta foi de se mostrar as informações das cartas, porém é a única parte que merece atenção neste trabalho. 

	
 
-API utilizada:

	API YuGiOh: https://db.ygoprodeck.com/api/v7/cardinfo.php?name=Dark%20Magician
